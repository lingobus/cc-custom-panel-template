A CocosCreator Custom Panel Template
====

Create a global package
====
```bash
#init project
cd ~/.CocosCreator/packages
vue init lingobus/cc-custom-panel-template <project-name>

#code <project-name>
#st <project-name>
#vim <project-name>
```

Open panel by click `Menu/Extension/<project-name>`

# Tips for panel developers:

 - Open Devtool for panel window by press `Cmd+Alt+I` on panel
 - Reload panel by press `Cmd+R` on panel
 - Reload package by press `Cmd+R` on Cocos Creator or click `Menu/Developer/Reload`
